# Prompt Engineering for Data Visualization

This repository contains the artifacts and resources used in the study of prompt engineering applied to data visualization.

## Overview

The goal of this project is to investigate how prompt design influences the generation of visual representations from structured data.

All experimental artifacts are publicly available to ensure transparency and reproducibility.

## Experimental Setup

The experimental configuration was designed to ensure consistency and reproducibility across all experimental conditions.

Four main components were established:

1. Reference images  
2. Dataset  
3. Visual configuration files  
4. Initialization prompts  

## Reference Images

A collection of images used as visual references to guide the generation process.

## Dataset

The dataset consists of structured data instances used as input for visualization generation tasks. It was curated to ensure consistency across experimental conditions and to support reproducibility.

## Visual Configuration Files

Configuration files that define how visual elements are structured and rendered during the experiments.

## Initialization Prompts

A set of prompts designed to initialize and guide the generation process.
